# Contributing
Thank you for your interest in contributing to this project.

When contributing to this repository, please first discuss the change you wish to make via issue, or any other method with the owners[^1] of this repository before making a change.

## Commit Messages
All commits should have adequate and suitable names.

Recommended format:
```
type: short description

long description (if required)
```
+ We prefer lower-case commit messages: `fix: ...` over `Fix: ...`.
+ Dots at the end of sequences in the commit is **NOT** recomended, but if you do place them, we don't care. 

[^1]: Developers is presented in [this file](./MEMBERS.md).
